## Python 패키지 requirements.txt 이용 방법

매번 Push하기 전에 패키지 정보를 저장하고 넘겨야 합니다. 본인의 가상 환경을 구축하고 Clone해야 이용할 수 있습니다.

```bash
pip freeze > requirements.txt
# 패키지를 버전 정보까지 함께 저장함.

pip install -r requirements.txt
# 패키지 설치
```

## 설치한 패키지 목록

- discord.py
- python-dotenv
- asyncio
